Web Application Project

Rate My Derp!

RateMyDerp.com is a web application which is an homage to the "Rating Sites" of the early 2000's (HotOrNot, RateMyFace, etc.). It's a full stack web application built using Python (Django), JavaScript, Sass and HTML. The site currently has over 1000 users, with more than 500 user submissions.
